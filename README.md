## 安装

```
npm install
```

## 运行

```
npm run dev
```

## 构建

```
npm run build
```

## 注意事项

- 通用组件中 `less` 不允许重名
- 通用组件放 `components` 文件夹目录下
- 首页以瀑布流布局的方式展示组件，目录路径是：`src/pages/index.tsx`
- Guide 组件展示组件的使用示例，目录路径是：`src/pages/index.tsx`
